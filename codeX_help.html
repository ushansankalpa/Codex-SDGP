Sign Master is a American Sign Language translator web application for users who's in need of understanding American Sign Language. Sign Master provides following features to it's users.


1) Live Video


User can choose this option to translate a live video from the camera feed


2) Upload Video


User can choose this option to upload a video from the user's media storage and translate


3) Sign Dictionar


Users who's in need to learn American Sign Language can choose this option to learn basic alphabetical signs
